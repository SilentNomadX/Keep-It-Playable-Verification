# Verification Checklist

## Hardware Checks
- Powers on/off correctly
- No overheating after 30–60 mins
- Fan noise within normal range
- HDMI output stable

## Function Checks
- Reads discs correctly
- Connects to internet / PSN
- Controller sync works
- USB ports working

## Service Actions
- Internal clean completed
- Dust removed from fan and heatsink
- Thermal paste replaced (if applicable)

## Final Status
- Pass / Fail
