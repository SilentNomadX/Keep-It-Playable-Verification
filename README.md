# Keep It Playable — Verification System

A practical system for testing, refurbishing, and verifying video game hardware.

## Purpose
To create a trusted standard for refurbished consoles, with proof of condition, functionality, and service history.

## What "Verified" Means
Each console is:
- Fully tested for core functionality
- Internally cleaned
- Assessed for performance (heat, noise, stability)
- Documented with evidence (photos/video)

## Current Status
Prototype stage — building and refining the verification process.
